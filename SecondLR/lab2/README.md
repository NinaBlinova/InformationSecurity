# Лабораторная работа №2 — Генератор-мультиплексор Дженнингса (вариант 3)

## Структура
- `jennings.py` — LFSR и генератор Дженнингса, функции `encrypt`/`decrypt`.
- `tests.py` — 5 статистических тестов (частотный, последовательный, серий, автокорреляция, универсальный Маурера).
- `main.py` — пример шифрования и запуск тестов.

## Как запускать
1) Активируй виртуальное окружение `venv` (из корня проекта):
   ```bash
   venv\Scripts\activate
   ```
2) Перейди в папку работы и запусти демонстрацию:
   ```bash
   python main.py
   ```

## Что делает `main.py`
- Шифрует строку «Привет, мир! Это тест генератора Дженнингса.»
- Расшифровывает и сравнивает с исходником (должно совпасть).
- Генерирует 20 000 бит ключевого потока и прогоняет 5 тестов.
- В выводе покажет число нулей/единиц и p-value для каждого теста.

## Параметры генератора
- РСЛОС A: степень 10, полином `x^10 + x^7 + x^5 + x^2 + 1`, начальное состояние `STATE_A` в `main.py`.
- РСЛОС B: степень 11, полином `x^11 + x^2 + 1`, начальное состояние `STATE_B` в `main.py`.
- Алгоритм: выходной бит = текущий бит B (до тактирования); A тактируется всегда; B тактируется 1 раз, либо 2 раза, если бит A == 1.

## Быстрый результат (по умолчанию)
- Пример запуска даёт равномерный поток (~50/50) и проходит все 5 тестов.

## Как использовать в своей задаче
- Для другого текста — поменяй строку `message` в `main.py`.
- Для другого объёма тестирования — измени `SEQ_LEN` (кол-во бит) в `demo_statistics`.
- Можно задать свои полиномы/состояния через `TAPS_A`, `STATE_A`, `TAPS_B`, `STATE_B`.

## Проверка корректности
- После запуска увидишь: шифртекст (hex), расшифрованный текст, совпадение /, статистику тестов с p-value.

## Мини-справка по тестам
- **Частотный**: равновесие 0/1.
- **Последовательный**: равновесие пар (00,01,10,11).
- **Серий (Runs)**: количество серий 0/1.
- **Автокорреляция**: зависимость соседних битов (lag=1).
- **Универсальный Маурера**: меряет сжимаемость; высокий p-value -> труднее сжать.
